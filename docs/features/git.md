# Git Integration

PyTPO includes common Git workflows in the `Git` menu.

## Available actions

- `Clone Repository...`
- `Commit...`
- `Push...`, `Fetch`, `Pull`
- `Preflight Check...`
- `Commit and Push...`
- `Branches...`
- `Rollback -> Discard Unstaged Changes...`
- `Rollback -> Hard Reset to HEAD...`
- `Refresh Git Status`

## Operational notes

- Most actions require the open folder to be a Git repository.
- Remote actions require valid credentials.
- Use `Refresh Git Status` if state appears stale.
