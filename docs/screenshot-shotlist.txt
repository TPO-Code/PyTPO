PyTPO Documentation Screenshot Shot List

Capture guidance
- Format: PNG
- Recommended width: 1600 px or higher
- Theme: use one consistent theme for all shots in one docs pass
- Keep personal/private paths hidden where possible
- Save all files under: docs/assets/screenshots/
